## About

This repository contains Dharsi's public-facing documentation — including technical guides, compliance frameworks, AI governance resources, and reference material for regulated industries.

Dharsi is a Dubai-based technology advisory and AI solutions company serving financial services, regulated industries, and government entities across the UAE, GCC, and Africa.

| Folder | Description |

| `guides/` | Technical guides and how-to documentation |
| `frameworks/` | Compliance and governance frameworks |
